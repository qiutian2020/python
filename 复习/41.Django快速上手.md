解决：
```shell
sudo yum install python3-devel
export C_INCLUDE_PATH=/usr/include/pythonX.X:$C_INCLUDE_PATH
```
确保将X.X替换为你安装的Python版本号。此命令将告诉编译器在这个路径下查找Python.h文件。
